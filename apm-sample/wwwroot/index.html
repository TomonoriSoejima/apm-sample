Welcome to APM Sample

This is a sample application that interacts with Elastic APM (Application Performance Monitoring). It helps developers monitor the performance of their applications, track transactions, capture errors, and analyze distributed traces.

The navigation links above allow you to explore different routes and test how APM logs requests and performance metrics in real-time.

You can use Elastic APM to:

  • Monitor request performance
  • Detect and debug errors
  • Analyze database queries and external API calls

Try clicking different links and check the APM dashboard in Kibana to see real-time monitoring!
